# Social-Sparks
The Social Spark App is a user-friendly Android application developed using Kotlin. Its purpose is to generate simple social interaction suggestions based on the time of day entered by the user.
A native Android application built with Kotlin in Android Studio that helps users maintain social connections by suggesting small, fun interactions based on the time of day. The app uses simple input logic (if/when statements) to provide personalized “sparks” such as sending a good morning text, sharing a meme, or leaving a thoughtful comment.
